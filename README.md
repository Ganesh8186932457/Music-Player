# 🎵 Simple Music Player

A lightweight and responsive web-based music player built using **HTML**, **CSS**, and **JavaScript**.

## 🚀 Features

- Play / Pause music
- Audio progress bar with seeking
- Responsive and minimal UI
- Easily customizable with your own audio tracks
